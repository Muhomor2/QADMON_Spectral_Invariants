# QADMON Spectral Invariants — Release v1.0.0

This release contains the complete reproducible package for the
QADMON Spectral Invariants project, including:

- Two new spectral invariants (kappa, gamma_star)
- Rigorous theorems (Theorems Companion)
- Main article describing the mathematical framework
- Python code for full reproducibility
- Numerical datasets (kappa convergence, gamma_star results)
- Zenodo metadata and CITATION file

## v1.0.0 — Initial public release (Zenodo archived)

- First publication of the spectral invariants γ* and κ.
- Includes article, theorem companion, Python code, and numerical data.
- Archived on Zenodo: 10.5281/zenodo.17788901.
